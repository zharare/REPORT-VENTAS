# CONSITEC - Registro de Ventas

CRM interno estilo SaaS construido con Next.js App Router, Tailwind CSS, Zustand, Recharts, dnd-kit y persistencia local vía LocalStorage.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`

## Funcionalidades

- Dashboard con KPIs y gráficos.
- Pestañas dinámicas por asesor con drag & drop.
- Tabla editable con 50 filas iniciales por asesor.
- Filtros por mes, año y búsqueda global.
- Persistencia completa en LocalStorage.
- Exportación a Excel de la pestaña activa.
